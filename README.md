# Ďîśʜà — موقع تفاعلي

ملف بسيط لعرض ملف تعريفي يتفاعل مع حركة الماوس.

تشغيل محلي:

1. افتح الملف [index.html](index.html) في متصفح حديث (متصفح يعمل بدون خادم مطلوب).

تخصيص:
- استبدل `avatar.svg` بصورة خاصة بك بنفس الاسم أو عدل المسار في `index.html`.
- لإضافة GIF كخلفية: ضع الملف باسم `bg.gif` في نفس المجلد (d:/website) أو غيّر المسار داخل `styles.css` للملف الذي تريد استخدامه.
- الكارد يعكس الخلفية عبر تأثير زجاجي (backdrop-filter). بعض المتصفحات قد لا تدعم `backdrop-filter` بالكامل — النتيجة الأفضل في Chrome وEdge.
- عدل النصوص في `index.html` لتغيير الاسم والوصف.
